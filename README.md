# gerenciador-de-notas

Um aplicativo de anotações **Desktop**, **Offline-first** e focado em produtividade para **desenvolvedores e estudantes**.

Desenvolvido com **Electron**, o app permite criar uma **base de conhecimento pessoal conectada**, com suporte avançado a Markdown, código, matemática e mídia — tudo armazenado localmente.

---

## Visão Geral

O **Notas de Pensamento** nasceu como um simples editor web e evoluiu para uma ferramenta completa de organização de ideias, estudos e projetos, priorizando:

- Controle total dos seus dados  
- Funcionamento 100% offline  
- Escrita fluida e bem formatada  
- Conexão entre ideias (estilo wiki pessoal)

---

## Funcionalidades

### Edição e Formatação

- **Markdown Nativo**  
  Escreva livremente usando a sintaxe padrão de Markdown:  
  `# Títulos`, `**negrito**`, `- listas`, `> citações`, etc.

- **Destaque de Sintaxe para Código (Highlight.js)**  
  Blocos de código são detectados automaticamente e renderizados com o tema **Atom One Dark**, ideal para armazenar snippets de:
  - JavaScript
  - Python
  - C / C++
  - Java
  - e outras linguagens

- **Matemática com LaTeX (MathJax)**  
  Suporte completo para fórmulas matemáticas complexas:
  ```latex
  $$x = y^2$$
